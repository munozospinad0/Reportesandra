# Reporte de Resultados — Sandra Clavijo

Presentación narrada e interactiva de resultados (marketing, calidad de leads, medición y mejoras del sitio) para la firma **Sandra Clavijo Immigration Law**.

## Cómo verlo
- **En vivo:** GitHub Pages (ver la pestaña *Settings → Pages* del repo).
- **Local:** abre `index.html` en cualquier navegador. Es **auto-contenido** (voz David + música + datos embebidos), no necesita internet ni archivos extra.
- Pulsa **"Reproducir el reporte"** y deja que avance solo. Controles: `←` `→` (navegar), `▶` (narrar), `F` (pantalla completa).

## Cómo se genera
- `_build_report.mjs` — arma `index.html` desde las slides (marca, gráficos, animaciones).
- `_voz_report.mjs` — narración con ElevenLabs (voz David), sync palabra-por-palabra, audio + música embebidos en base64.

Datos reales de CRM (Neon), Google Ads, GA4 (propiedad 513029805) y GTM.
